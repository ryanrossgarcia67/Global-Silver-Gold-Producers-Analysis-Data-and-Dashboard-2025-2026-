# Global-Silver-Gold-Producers-Analysis-Data-and-Dashboard-2025-2026-
## Overview
Interactive Power BI dashboard analyzing global silver mining companies across production, cost efficiency, and metal exposure.

## Key Features
- Standardized inconsistent data (GEO vs oz, mixed formats)
- Compared production vs margin across companies
- Visualized gold vs silver exposure

## Tools
Power BI, DAX, Data Cleaning

## Insights
- Larger producers tend to show stronger cost efficiency
- Clear segmentation between silver-focused vs diversified miners
